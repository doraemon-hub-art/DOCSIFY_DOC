# 前言

18岁后，似乎时间被按上了倍速键，当我反应过来时，已经即将24了，转过头想一想，这六年里也发生了这么多事。从校园到社会，从实习到正式，我也换了三家公司，在外租房也搬过了四次，认识了很多人，发生了很多事，经历了很多，也学会了很多，但总是自己始终没有取得一些令自己满意的成绩。

工作上也好，生活上也罢，或是自己的一些兴趣爱好。有的没有去坚持，有的没有去做，我一次又一次地尝试浮上水面，一次又一次的沉入海底，虽然没有达到目标，但还算幸运的是，我总有起伏的勇气。

2025也不例外，要说跟往年完全不同的，今年是我完完全全开始工作进入到社会的一年。我觉得从今年开始，或多或少，我的心态比以往更加成熟了。

在这里洋洋洒洒的，用流水账记录一下这精彩2025吧，这是我这几年写总结最早的一次。

我没有翻过那座山，但也并非空手而归。

---

# 平静而沉寂的日子

下半年那会儿，想把锻炼重新拾起来，就在出租屋那边，也添置了一些器材，动感单车、引体架子、哑铃凳，还有一些哑铃。可以说有氧和力量，都挺齐全了。

除此之外，在八月份找工作那会儿，将Docsify的笔记分站重新搭起来了，这四个月过去，慢慢积累了一些东西，也画了很多大饼，想了很多想做的视频内容，不过目前只在国庆期间出了一起Markdown的入门视频。

进行这种内容输出，可以给我很大的正向反馈，即便，现在几乎就没有人看我的内容。好像我一定要找点“正事”做，内心还比较能得到满足。

正如我在笔记分站写下的，“在世界上的一个不知名角落里，一颗希望的种子，已被种下。”，我不期待能够结出硕果累累，只希望在我闲暇之余，往树上望去，能够在层叠的叶子中，找到一颗甜甜的果实。

关于视频内容，我总是在想，东做一块，西做一块，这种混杂且平庸的内容，很难吸引具有粘性的观众，在这条创作之路，我还有很长的路要走。

哔哩哔哩做一些内容的长视频，抖音发一些日常 + 一些游戏剪辑，或是我一直在酝酿但是没有学成的吉他小视频。

在这个思考的过程中，一句反复让我犹豫的话是，一个已经有人做出来的东西，自己是否还有必要做一遍？

# 被推着走，跟着生活流

今年上半年的主要基调就是出差，从三月份开始直到七月初，陆陆续续地出差才彻底结束，从苏州街渠道苏州长三角研发中心、苏州相城区总部、河北固安，感觉出差了六七回不只，最晚的一次，在苏州干到了凌晨三点。有那么一段时间，一到周五我就很烦，因为不知道几点能下班，这周什么时候可以休息。

虽然说工作上的一些折腾令人没那么开心，但是跟下面一起干活的同事们合作，还是令人十分愉悦的。大家都很友好，出差在固安的那段时间，我们晚上天天出去吃顿好的。

为了这个项目，很长一段时间内，几乎都不怎么在公司办公，我们一起去利星行外勤挤在一个小屋中，去固安出差不是车上就是地库里，去苏州长三角、公司总部，地库车间东奔西跑，纵使辛苦，现在回想起来，也有很多美好有趣的回忆。

可故事的过早结束，并不在我的预料之内。

从上半年开始，同项目组的小伙伴就陆陆续续的离开了，或是主动，或是被动，直到八月底，我熟悉的同事已经走了很多了，在这期间，我也无法幸免。我不得不离开，不得不找出去闯闯，重新找回自己。

于是我像本科校招那会儿，改简历准备面试，有了一些工作经验后，稍微弥补了一些我学历上的弱势，比那会儿校招给机会的公司多，也面了几个大厂，面了开始没兴趣的小公司，面了有兴趣还不错的公司，有的说我基础太差，有的说我不错，有的说我有长版但是与岗位不太符，有的一面挂了，有的二面挂了，有的候选人比对挂了。

就好像命运一直以来对我的态度一样，我的运气总是没那么好，但总归会留给我一丝喘息之地。于是，我收拾行囊，重新看向远方，开启新的征程。

这是我今年最大的变动，我离开了从23年实习开始的公司，离开了在校园与社会之间的中转站。离开了熟悉的环境，不错的同事。在最后的那段时间，我很痛苦，很纠结，对未来的不确定，对故人的不舍，同时也激起了我一些不应该的情绪，那段时间我很难受，吃饭也没啥胃口，偶尔回想起来，就像是最后高考那段时间里一样，我总是尽量避免把这段不好的经历在我的脑海中在重新播放。与现在已经过去了相比，似乎我好像就根不相信这事儿发生我我身上一样。

老前辈说我的是第一次跳槽，想的太多。确实，想的很多，总是瞎想。说我还年轻，有试错机会，可以出去闯闯，在这也没发展。

很幸运能在进入社会的第一站就遇到这些很好的人，指引方向的前辈、友好的玩伴、不错的同事。我是项目组最小的，大家也很照顾我，有一次大家聚餐时聊天，超哥说我跟她表妹差不多大，都挺小的。早晚有一天，我也不会是公司最小的，就像是亮哥说的，开始工作那会儿，都管别人叫哥，后来慢慢的变成哥了，再后来都叫叔了。

那段时间，跟大家聊了很多，亮哥、jiying、磊哥、超哥、年哥、会哥、晓静姐、饼哥、fan哥等。

前两天回去吃亮哥的散伙饭，饭桌上跟大伙聊天的那一刻，好像大家都没走，还是那个完整的I3团队。吃完饭跟年哥闲聊了两句，跟大家分道扬镳，重新回到了新的归处。

故事的最后，不太完美，但我还算潇洒地离开。

我比较认生，适应新环境的能力还行，但是适应新人的能力不太行。跟新认识的同事，我不太好建立一些熟悉的关系，不过慢慢的总会熟悉就是了。

而且这边，也比我想象的要好一些，大家也都不错，一个更年轻的团队，工作上，我也有了些更大的发挥空间，只是，我目前还不能完全施展拳脚。

# 进一步

持续的输入与输出，平静的心态和坚持的态度，将是我未来的核心基调。

持续的输入，可以给我提供更多的灵感与内容，而有的灵感转瞬即逝，必须要抓紧时间记录下来，以及最后输出成内容。在今年，我会以视频形式，尝试做各种类型的内容。

有些要做的事，一定要做，有些需要做出的改变，也同样无法避免，或许以三到五年的长远视角考虑，一切都还来得及。

只是不知道何时才能改变当下不满的一切，反复地充满激情又失落，在不甘中找寻自我。

平静下来，着眼于自身，努力走的更远一些吧。
