# 文章格式设置

## 文本粘贴

从VSCode中，Ctrl + Shift + V —— 这个是无格式粘贴（粘贴为纯文本，去除原格式），这样可以将文章内容，格式完全一致的复制过去。

## 图片

> 大小

- StackEdit编辑器不支持MD全局设置IMG图片格式，需要单独图片设置，例如:

```bash
<img src="https://oss.banshengua.top//blogimages/20251221183159346.png"      width="30%"      style="display:block;margin:0 auto;">
```

- 但是Docsify中是支持文章内全局格式设置的，例如:

```bash
# 可以在文章开头设置，这里忘了，没找到之前的记录。。。 TODO: 找到了补上
```

其中:

- 30% 适合手机竖拍图片；
- 40% 适合横屏图片；
